Comment fonctionne le Web et ce qu'est une page Web

le Web permet visitez des sites mises en lignes, il suffit d'avoir une connection internet et de tapez le nom du sit ou l'url. Une page web est une page qui peut être visitée à l'aide d'un navigateur

Les rôles des langages HTML et CSS dans une page web

HTML et CSS jouent un rôle primordial dans la création d'une page web. HTML permet d'ecrire de texte, d'ajouter des images, des liens et d'autres, CSS de son côté est là pour styliser la pa web, donner de couleur, bien arranger les éléments dans la page web
